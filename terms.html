📄 Términos de Servicio

Esta aplicación fue desarrollada con el propósito de conectar con la API de TikTok Ads para obtener, analizar y visualizar datos de campañas publicitarias autorizadas.

El uso de esta aplicación está restringido a fines internos y no comerciales. Los datos accedidos mediante la API de TikTok se utilizan únicamente con propósitos analíticos.

No se recopila información personal de usuarios externos ni se comparten datos con terceros.

Al utilizar esta aplicación, aceptas estos términos de servicio y el tratamiento responsable de los datos.
